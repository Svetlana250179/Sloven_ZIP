2. sklon — KOGA? ČESA?
Moški (živo + neživo) adj: -i → -ega
sam: osnova → -a
Ženski (-a osnova) adj: -a → -e
sam: -a → -e
Ženski (soglasnik) adj: -a → -e
sam: osnova → -i
Srednji adj: -o → -ega
sam: -o → -a
IZJEME: drevo→drevesa, ime→imena
5. sklon — O KOM? O ČEM? (z predlogom)
Moški (živo + neživo) adj: -i → -em
sam: osnova → -u
Ženski (-a osnova) adj: -a → -i
sam: -a → -i
Ženski (soglasnik) adj: -a → -i
sam: osnova → -i (= rodilnik!)
Srednji adj: -o → -em
sam: -o → -u
IZJEME: drevo→drevesu, ime→imenu
